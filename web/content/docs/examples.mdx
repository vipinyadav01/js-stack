# Examples

Real-world examples of projects you can create with JS-Stack CLI.

## SaaS Application

A complete SaaS application with authentication and database:

```bash
npx create-js-stack@latest my-saas \
  --frontend nextjs \
  --database postgresql \
  --orm prisma \
  --auth better-auth \
  --addons docker,testing \
  --package-manager pnpm \
  --git \
  --install
```

**What you get:**
- Next.js full-stack application
- PostgreSQL database with Prisma
- Better Auth for authentication
- Docker configuration
- Testing setup
- Git initialized

## Quick Prototype

For rapid prototyping:

```bash
npx create-js-stack@latest my-prototype \
  --frontend react \
  --backend express \
  --database sqlite \
  --orm prisma \
  --package-manager npm \
  --yes
```

The `--yes` flag skips all prompts and uses sensible defaults.
